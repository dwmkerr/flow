# flow

Quickly look up keyboard commands to maintain flow.

## Quickstart

Flow helps you quickly look up keyboard shortcuts without breaking your flow. It reads shortcuts from a markdown file and displays them in an easy-to-read format.

### Installation

```bash
# Clone and open the repo.
git clone https://github.com/dwmkerr/flow.git
cd flow

# Initialise a new flow-file and install the flow command.
make init
make install

# Try it out!
flow
```
